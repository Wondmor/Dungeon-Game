这是一个2D类挺进地牢的游戏demo

实现的功能：

1.游戏3C：
(1)Character：玩家动画，枪械瞄准鼠标位置，开枪时后坐力
(2)Camera: 跟随玩家移动，开枪时抖动
(3)Controller： 玩家移动，开枪，开宝箱，捡物品，买道具

2.UI系统，包括血量，护盾，枪械，子弹数量

3.商店和金币系统

4.对象池系统优化物品生成

5.使用RuleTile，使得放置普通地块能够自动变化边缘为设定的地块，加快地图编辑
